# Genc App
Search for your car by color and price.

* UI
    * Compose
    * Material design
    * Dark/Lite theme

* Tech stack
    * Kotlin
    * Coroutines
    * Hilt
    * Navigation
    * ViewModel
    * Retrofit
    
* Modern Architecture
    * Single activity architecture
    * MVVM for presentation layer
    * Domain layer for business logic
    * UI Layer is exposed with StateFlow
    * Android Architecture components
